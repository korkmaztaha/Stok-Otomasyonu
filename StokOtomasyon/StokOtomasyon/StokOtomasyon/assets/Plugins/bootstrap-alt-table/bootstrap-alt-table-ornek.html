An Alternative to Bootstrap Responsive Tables

This example demonstrates an alternative approach to responsive tables than what Bootstrap offers. A Bootstrap 3 responsive table will introduce a horizontal scroll bar that allows you scroll to the right to see the remaining content. The CSS in this example stacks the table columns vertically instead for each table row, which is easier to read on devices in portrait mode. Clicking the button below will toggle between the Bootstrap CSS and my CSS. Resize your browser to a size less than 768px wide and use the button below to toggle the effects.
